Welcome to slider Ball Game

Game Rules :

  • Use the slider to choose a number
  • keep the slider to the number you want
  • If the Number you choose greater then the random number then ball turns green
  • If the Number you choose lesser then the random number then the ball turns red
  • If the Number you choose is the same as the random number then ball turns blue and you won!
  • You have only three attempts to play this game
